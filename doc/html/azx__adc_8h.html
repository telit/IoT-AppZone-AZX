Detailed Description

Read from and write to a peripheral via ADC.

Version
1.0.2
Dependencies
core/azx_string core/azx_ati
Author
Ioannis Demetriou
Sorin Basca
Date
10/02/2019

This library makes it easy to write and read ADC values from a GPIO pin. This is achieved by using AT commands.

Warning
There is no mechanism to set an ADC value.
